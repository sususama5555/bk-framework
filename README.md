# :wrench: 前言

基于蓝鲸开发框架2.0，开发了集成三种功能的SaaS	~~（为考试做准备）~~

蓝鲸开发框架2.0使用说明：https://docs.bk.tencent.com/blueapps/USAGE.html

**`后端`**：`Django` + `MySQL` + `RabbitMQ` + `Redis`

**`前端`**： `Vue.js` + `Django Template`

**`组件`**：`element-ui` + `Echart` + `axios`

# :art:功能

- **文件备份与记录**
- **主机性能监控**
- **模板与任务**
